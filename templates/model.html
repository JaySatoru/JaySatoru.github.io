Deep Learning Model Architecture

Multimodal AI for Cystitis vs Bladder Cancer Classification

1. Image Branch

We used EfficientNetB1 pretrained on ImageNet for extracting high-level image features from bladder scans. Fine-tuning was applied in stage 2 to adapt the model for medical imaging.

Transfer Learning significantly improved performance with limited medical data.

2. Clinical Branch

Clinical features such as Age, Creatinine, GFR, Smoking History, Hematuria, etc., were processed using fully connected dense layers.

Feature scaling and normalization were applied before model training.

3. Multimodal Fusion

Image features from EfficientNetB1 and structured clinical features were concatenated and passed through dense layers to produce final classification.

Multimodal learning improves diagnostic robustness and reduces misclassification.
